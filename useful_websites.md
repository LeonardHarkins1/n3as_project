# Also the websites I recommended were:
* arxiv.org : online repository of “pre-prints”, which typically are the final draft of scientific paper before submission to a journal.
* http://inspirehep.net data base, which lists the contents of the arXiv, but has a nicer user interface, with more filter options, information about authors, jobs, conferences, etc. For example it links papers reporting some experimental observation to the corresponding PDG (particle data group) page.
* https://pdg.lbl.gov database of the particle data group, it contains listings for subatomic particles properties with references to the journal article where the measurement was reported. It also contains reviews for several particle physics topics, although most of them assume a graduate level of physics knowledge, but it still could be a handy reference if you explore its multiple reviews.

Finally, you should try to reread the two nature papers, and begin compiling some notes/summary for yourself, adding some references, which you can take directly from the papers. It is a good exercise that allows you to see how much you really are following from the readings. See you next Wednesday at 9:30am.
